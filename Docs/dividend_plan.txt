DIVIDEND TRACKER — IMPLEMENTATION PLAN
=======================================

CONTEXT
-------
Add dividend tracking to the Opes stock analysis app in two places:
  1. Dedicated "Dividends" tab — full dividend profile (yield, payout ratio,
     history chart, safety score) + a "Top Dividend Stocks" screener from the
     36-stock watchlist
  2. Dividend card in Analyze tab — compact flip card showing basics when any
     stock is analyzed, alongside the existing RSI/MACD/Bollinger cards

Goal: widen appeal beyond growth/momentum traders to dividend-income investors.

KEY DISCOVERY FROM EXPLORATION
--------------------------------
- analyze.py already requests summaryDetail and calendarEvents quoteSummary modules
- It already returns dividend_yield but ignores dividendRate, payoutRatio,
  exDividendDate — those fields are in the already-fetched response, just not parsed
- Dividend history requires events=dividends on the chart API call (not currently used)
- ~15 of 36 watchlist stocks pay meaningful dividends:
  XOM, CVX, COP, KO, PG, WMT, JNJ, ABBV, JPM, BAC, CAT, HON, DE, RTX, UNH


DIVIDEND SAFETY SCORE (0-100)
------------------------------
New shared function: compute_dividend_safety_score() in api/utils/scoring.py

Factor                              Condition         Points
----------------------------------  ----------------  ------
Payout ratio                        <40%              +30
                                    40-60%            +20
                                    60-75%            +10
                                    >75%              -10
                                    >100%             -30
FCF coverage (op CF / annual divs)  >3x               +25
                                    2-3x              +15
                                    1-2x              +5
                                    <1x               -20
3yr dividend growth                 Positive          +20
                                    Flat              +5
                                    Cut/eliminated    -30
Earnings growth                     Positive          +10
                                    Negative          -5
Beta                                <1                +10
                                    1-1.5             +5
                                    >1.5              -5

Score clipped to 0-100.
Labels: 80+ = "Very Safe", 60-79 = "Safe", 40-59 = "Moderate Risk", <40 = "High Risk"

3. api/dividends.py (CREATE)
   GET /api/dividends?symbol=X

   Fetches concurrently:
     - get_company_overview(symbol) — reuse same function from analyze.py
       (summaryDetail, financialData, calendarEvents, defaultKeyStatistics)
     - get_dividend_history(symbol) — NEW: chart API with events=dividends,
       range=5y, returns [{date, amount}] quarterly/annual payments

   Computes:
     - YoY dividend growth rate from history
     - Payment frequency (monthly/quarterly/annual from history interval)
     - Consecutive years of growth (dividend aristocrat indicator)
     - compute_dividend_safety_score(fund)

   Response shape:
   {
     "symbol": "KO", "name": "Coca-Cola Co.",
     "dividend_yield": 3.12, "dividend_rate": 1.94,
     "payout_ratio": 0.74,
     "ex_dividend_date": "2026-03-14", "next_dividend_date": "2026-04-01",
     "trailing_div_rate": 1.88, "payment_frequency": "quarterly",
     "consecutive_growth_years": 62, "growth_rate_3yr": 5.2,
     "safety_score": 78, "safety_label": "Safe",
     "history": [{"date": "2020-03-13", "amount": 0.41}, ...],
     "pays_dividend": true
   }

   Non-dividend payers: returns {"pays_dividend": false, ...} with nulls.

4. api/top_dividends.py (CREATE)
   GET /api/topdividends

   Processing:
     1. Concurrent fundamentals fetch for all 36 stocks
        (ThreadPoolExecutor, cap MAX_FUND_WORKERS=8 — reuse topstocks.py pattern)
     2. Filter: dividend_yield > 0.5%
     3. Compute compute_dividend_safety_score() for each
     4. Composite dividend ranking score:
          Yield score (yield mapped 0-100, capped at 6% max) x 0.4
          Safety score x 0.4
          Fund score (existing compute_fundamentals_score()) x 0.2
     5. Quick RF run for ML direction on top stocks
     6. Return top 10 sorted by composite score

   Response shape per stock:
   {
     "symbol": "ABBV", "name": "AbbVie Inc.", "sector": "Healthcare",
     "price": 165.20, "change_percent": "+0.45%",
     "dividend_yield": 3.8, "dividend_rate": 6.20, "payout_ratio": 0.55,
     "safety_score": 82, "safety_label": "Very Safe",
     "consecutive_growth_years": 52,
     "direction": "bullish", "rating": "Moderate Buy", "quality_score": 68.4
   }
